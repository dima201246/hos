Описание функций:

	- dlg_win	-	Функция, которая позволяет получить ввод от пользователя.
					Позволяет установить три кнопки.

					Передаваемые параметры:
						- DLGSTR	- Структура с параметрами окна.
						- string&	- Переменная, куда будет возвращён пользовательский ввод.

					Возвращаемые значения:
						- int		- Возврат самой функции, возвращает номер нажатой кнопки (от 1 до 3).
						- string&	- Передаётся как параметр функции, описание смотрите выше.

	- msg_win	-	Функция выводящая какое-то сообщение с одной кнопкой.
					Текст на кнопке можно поменять.

					Передаваемые параметры:
						- DLGSTR	- Структура с параметрами окна.

					Возвращаемые значения:
						- int		- Возврат самой функции, возвращает номер нажатой кнопки (от 1 до 3).

	- menu_win	-	Функция для вывода на экран какого-либо списка.
					Поддерживает прокрутку.
					Поддерживает возврат номера выделенного элемента.
					Кнопок нет.

					Передаваемые параметры:
						- DLGSTR&			- Структура с параметрами окна.
						- vector <string>&	- Список элементов для вывода (ВНИМАНИЕ! Смотрите пункт возвращаемые значения!).

					Возвращаемые значения:
						- DLGSTR&			- Передаётся как параметр функции, через структуру возвращается позиция конца выделения текущего элемента (xreturn, yreturn) (см. схм1)
						- vector <string>&	- Передаётся как параметр функции. Подвержен изменению со стороны окна, то есть, будет изменён в зависимости от размера окна (обрезаны некоторые строки и/или добавлены пробелы)

Структуры:

	DLGSTR:
		Тип string:
			title				-	Текст заголовка окна (не будет текста - не будет заголовка) (может быть пустым).
			line				-	Сам текст, который будет выводится (не может быть пустым.
			f_button			-	Текст на первой кнопке.
			s_button			-	Текст на второй кнопке.
			t_button			-	Текст на третьей кнопке.
	
		Тип int:
			num_of_chars		-	Максимальное число символов, которое можно ввести (пока не работает).
			type_input			-	Тип ввода (пароль, только буквы, только цифры и др.) (пока не работает).
			style				-	Стиль окна (см. описание стилей).
			keys				-	Количество кнопок в окне (от 1 до 3).
			fix_size			-

		Тип unsigned int:
			xpos, ypos			-	Координаты окна на экране.
			xmax, ymax			-	Максимальные размеры окна. Внимание!
									Свободное пространство для окна определяется как:
										*pos + *max [+ 2 (границы окна, если включены)] [+ 1 (загаловок, если включён, если это ypos)]
										(где * - x или y)
			selected			-	Номер выделенного элемента.
			xreturn, yreturn	-	Координыты указателя.
			second_border		-

		Тип bool:
			border_menu			-	Граница вокруг элементов окна.
			active_menu			-
			not_view_scrollbar	-	Не показываль скролл-бар, даже если элементы воходят за границу.

Схемы:

	схм1:
		--------------
		|Title       |
		]1 Item      | <- xreturn, yreturn
		|2 Item      |
		--------------
